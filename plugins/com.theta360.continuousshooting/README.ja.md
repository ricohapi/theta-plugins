## 説明

<div id="plugin-description">

プラグインを起動し、シャッターを押すだけで、撮影条件はAUTOで９枚、連続撮影を行います。撮影条件によっては、最短0.3秒で撮影を終えることができます。当プラグインをTHETAの「使用するプラグイン」に設定しておけば、スマートフォンの基本アプリを介することなく、本体操作で連続撮影ができて便利です。  

※撮影後、数十秒程度の処理時間を要します。

</div>
